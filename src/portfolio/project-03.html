A hobby project written in python to convert image files into ASCII art displayed to the console. Inspired by my love for art and code, I created this program to see the two worlds mesh. With an image file, some colourful ASCII escape codes, and some innovative characters, Art can be drawn to the console with the hit of the enter key.

Utilizing 28 possible characters to show brightness, and the full 0-255 rgb colour space, images are parsed and displayed to the console. An average box sampling algorithm rescales the image to the console's character resolution. The grayscale brightness of these new pixels is mapped to the character density scale, and an ASCII escape code is used to apply the colour.

To learn more, see more beautiful examples, and possibly try it yourself, check out my GitHub.
