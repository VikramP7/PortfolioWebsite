The Weather Tree

Using APIs to Make Art

  • Programming
  • Python
  • Hardware Interfacing
  • APIs
  • Algorithms
  • UI Design
  • Art

The Weather Tree utilizes lights and movement to create an artistic display of the weather conditions of various places in the world. The trunk and leaves of the tree light up with different colours signaling the temperature and animations play displaying the condition (rain, snow, lightning, etc). The golden orangutan points in the wind direction.

This was a first year eng. group project, I was responsible for the 3D design of the tree, wiring design, UI, backend, and animation development.

The guts of the project are written in micropython, on a Raspberry Pi Pico W. The Pico fetches and parses weather data from the Weather API and displays these values through led lights in the tree, the LCD display and the servo controlling the orangutan. I modeled the tree in Blender 4.0 and Fusion 360. Learn more about how it works on my GitHub!
