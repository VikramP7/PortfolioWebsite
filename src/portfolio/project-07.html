F17 Humanitarian Hummingbird Drone

Fully custom PCB drone from Specific Dynamics

  • Custom PCBS
  • BLDC Motor ESC
  • Custom Firmware
  • PID Control
  • Remote Control

The F-17 Humanitarian Hummingbird is a lightweight, fully modular quadrotor microdrone built from scratch as a six-person team for ENEL 400. Defined by its PCB-as-airframe architecture, where the electronic speed controller board doubles as the structural frame. The result is a 145 g, low-cost, hackable platform aimed at general-purpose UAV applications. My focus on the team was the custom brushless motor controller.

The ESC drives four three-phase motors from a single STM32F405 running bare-metal C. It implements sensorless six-step trapezoidal commutation, using back-EMF zero-crossing detection to track rotor position without hall sensors. All three on-chip ADCs are triggered in lockstep at 52.5 kHz to sample BEMF across every motor. The finished controller outperformed the commercial off-the-shelf ESC used as an early stand-in.

The rest of the system pairs an ESP32-S3 custom flight controller running a 1 kHz cascaded PID loop with Mahony 9-axis sensor fusion, and a handheld ESP32-C3 remote that streams pilot input over ESP-NOW. Across seven custom PCBs and over 3,000 lines of C, the drone achieves stable manual hover for roughly 5.5 minutes with usable control range beyond 20 m. To learn more about the project see the team GitHub.
